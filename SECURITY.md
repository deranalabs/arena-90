# Security Policy

## Reporting a vulnerability

Please do not disclose suspected vulnerabilities in a public issue.

Use GitHub's private vulnerability reporting for this repository:

1. Open the repository's **Security** tab.
2. Select **Advisories**.
3. Select **Report a vulnerability**.

Include the affected component, reproduction steps, potential impact, and any
suggested mitigation. Do not include private keys, seed phrases, wallet files,
provider credentials, or personal data in the report.

Arena90 currently operates its supporter lifecycle on Solana devnet. Do not
send mainnet assets to addresses or transactions referenced by this repository.

## Supported version

Only the current default branch is supported during the hackathon release.
Historical branches and archived deployments are unsupported.
